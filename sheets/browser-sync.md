### Installing browser-sync

```bash
npm i -g browser-sync
```

### Start a server

```bash
browser-sync start --server <path> --files='**/*.html, **/*.css'
```

### Options

```bash
  --port=N
  --proxy="http://127.0.0.1:3000"
```
