## Shortcuts

<!-- {.-three-column} -->

### Movements

| Key           | Description               |
| ------------- | ------------------------- |
| `C-n` / `C-p` | up/down                   |
| `C-f` / `C-b` | left/right                |
|               |                           |
| `C-v` / `M-v` | Page up/page down         |
|               |                           |
| `C-a` / `C-e` | Beginning/end of line     |
| `M-a` / `M-e` | Beginning/end of sentence |

### Basic

| Key         | Description |
| ----------- | ----------- |
| `C-x` `C-f` | Find file   |
| `C-x` `C-s` | Save file   |

### Command line

| Key   | Description       |
| ----- | ----------------- |
| `M-x` | Open command line |
