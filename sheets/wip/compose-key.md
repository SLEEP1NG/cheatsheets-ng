### Common symbols

| Character | Shortcut | Description     |
| --------- | -------- | --------------- |
| •         | `.=`     | Filled bullet   |
| ·         | `.-`     | Middle dot      |
| ×         | `xx`     | Multiplication  |
| ®         | `oR`     | Registered sign |
| ™         | `tm`     | Trademark sign  |
| ∴         | `:.`     | Therefore       |
| §         | `OS`     | Section sign    |
| ♥         | `<3`     | Heart           |
| →         | `->`     | Arrow           |
| ⇒         | `=>`     | Arrow           |
| ☺         | `:)`     | Smile           |
